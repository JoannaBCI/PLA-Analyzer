# PLA-Analyzer
Fluorescence microscopy tool for quantifying PLA puncta and protein signal per nucleus, grouped by cell type.

A Python tool for quantifying protein interactions from fluorescence microscopy images using Proximity Ligation Assay (PLA) data.

## What it does
- Segments nuclei from DAPI channel
- Counts PLA puncta (dots) per nucleus
- Groups results by cell type
- Outputs summary plots and CSV data
